# Bayesian Information Criterion

S3 method to compute BIC (Bayesian Information Criterion).

## Usage

``` r
# S3 method for class 'lifelihoodResults'
BIC(object, ...)
```

## Arguments

- object:

  output of
  [`lifelihood()`](https://nrode.github.io/Lifelihood/reference/lifelihood.md)

- ...:

  Ignored

## Value

The BIC
