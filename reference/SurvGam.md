# Gamma survival function

Gamma survival function

## Usage

``` r
SurvGam(t, expt_time_to_event, scale)
```

## Arguments

- t:

  Numeric. The time to event

- expt_time_to_event:

  Numeric. The expected longevity or time to maturity.

- scale:

  Numeric. The scale parameter

## Value

The survival probability (numeric)
