# MCMC

Retrieve the mcmc from the output of
[`lifelihood()`](https://nrode.github.io/Lifelihood/reference/lifelihood.md)

## Usage

``` r
mcmc(object)
```

## Arguments

- object:

  output of
  [`lifelihood()`](https://nrode.github.io/Lifelihood/reference/lifelihood.md)

## Value

A covariance matrix
