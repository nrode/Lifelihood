# Weibull survival function

Weibull survival function

## Usage

``` r
SurvWei(t, expt_time_to_event, shape)
```

## Arguments

- t:

  Numeric. The time to event

- expt_time_to_event:

  Numeric. The expected longevity or time to maturity.

- shape:

  Numeric. The shape parameter

## Value

The survival probability (numeric)
