# Log-normal survival function

Log-normal survival function

## Usage

``` r
SurvLgn(t, expt_time_to_event, vp1)
```

## Arguments

- t:

  Numeric. The time to event

- expt_time_to_event:

  Numeric. The expected longevity or time to maturity.

- vp1:

  Numeric. The vp1 parameter

## Value

The survival probability (numeric)
